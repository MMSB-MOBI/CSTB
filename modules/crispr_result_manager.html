CSTB result manager¶

class CSTB.crispr_result_manager.CrisprResultManager(pycouch_wrapper, taxondb, genomedb, motif_broker_endpoint, tag, include_taxon={}, exclude_taxon={}, nb_total_hits=None, nb_treated_hits=None, hits_collection=[], homolog_genes=[])¶

Bases: object

Manager for crispr results

Attributes:

wrapper (pycouch.wrapper.Wrapper): pycouch wrapper to interrogate couchDB motif_broker_endpoint (str): motif-broker endpoint to get sgRNA occurences from couchDB with mapping taxondb (str): taxon database name genomedb (str): genome database name tag (str): tag for result storage nb_total_hits (int): Number of hits found by setCompare. Set with parse_set_compare() call. nb_treated_hits (int): Number of setCompare hits treated by this manager. Set with parse_set_compare() call. hits_collection (List[Hit]) : List of Hit objects include_taxon (Dict) : Dict with included genomes uuid as keys and corresponding taxon name as value exclude_taxon (Dict) : Dict with included genomes uuid as keys and corresponding taxon name as value

parse_set_compare(setCompare_file, word_length, to_keep)¶

Parse results from setCompare. Will call a different parsing function for word with size 20 and for shorter words. Separate functions are required because setCompare results format is not the same with with 20-length words and shorter words. Initialize hits_collection and nb_treated_hits

Parameters

  • setCompare_file (str) – path to setCompare result file

  • word_length (int) – Word length

  • to_keep (int) – Number of hits to keep

parseBlast(blast_xml, identity, included_genomes)¶

Take blast xml file and parse it. Just keep homolog genes with more than defined identity and just keep coordinates that maps with homolog genes.

Parameters

  • blast_xml (str) – path to blast xml result

  • identity (int) – minimum identity percentage

  • included_genomes (List[str]) – list of included genomes uuid

Raises

  • error.NoBlastHit – Raise if there is no blast hit at all

  • error.NoHomolog – Raise if at least 1 included genome has not an homolog gene.
